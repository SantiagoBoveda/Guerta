Configuración
Conexión a la nube · sincronización · acceso desde celular
📱 Instalar en el celular

Güerta es una PWA — se puede instalar en el celular como si fuera una app nativa, sin pasar por ninguna tienda.

🤖 Android (Chrome)
  1. Abrí la app en Chrome
  2. Tocá el menú ⋮ (tres puntos)
  3. Seleccioná "Agregar a pantalla de inicio"
  4. Confirmá → aparece el ícono de Güerta
🍎 iPhone (Safari)
  1. Abrí la app en Safari
  2. Tocá el botón compartir ↑
  3. Seleccioná "Agregar a pantalla de inicio"
  4. Confirmá → aparece el ícono de Güerta
⚠ Importante: Para que ambas socias compartan los mismos datos, primero configurá Supabase abajo. Sin eso, cada celular tiene su propio inventario independiente.
☁️ Supabase — base de datos compartida
Una cuenta gratuita alcanza. Ambas socias se conectan al mismo proyecto.
Sin configurar
Pasos para configurar (5 minutos):
1. Entrá a supabase.com y creá una cuenta gratuita
2. Creá un nuevo proyecto (elegí región más cercana: South America)
3. Andá a Project Settings → API
4. Copiá la Project URL y la anon public key
5. Pegalas abajo y tocá Conectar
6. En el SQL Editor de Supabase, ejecutá:
CREATE TABLE guerta_snapshot ( id TEXT PRIMARY KEY, data TEXT NOT NULL, updated_at TIMESTAMPTZ DEFAULT now() ); ALTER TABLE guerta_snapshot ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_access" ON guerta_snapshot FOR ALL USING (true) WITH CHECK (true);
